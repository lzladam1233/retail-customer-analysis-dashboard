**Customer Purchase Behavior Dashboard (Power BI)**
📊 **Overview**

This project analyzes customer purchasing behavior using Power BI. The dashboard provides insights into revenue trends, customer demographics, and category performance.

🚀 **Key Insights**

Total revenue: $233K
Clothing category generates the highest revenue
Average customer spending: $59.76
Seasonal trends impact purchasing behavior
📈 Features
Interactive filters (Gender, Category, Location, Season)
KPI cards for business overview
Customer segmentation by purchase frequency
Category-level performance analysis
